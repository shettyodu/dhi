How the program works

3 · Earn

Recommended payout

7% of gross profit on each completed sale attributed to your referral code. Alternatives we can discuss: a flat fee per video/post, or a hybrid (small upfront + performance bonus). Final terms are set in your partnering agreement.

Promotion requirements

  • • Clearly disclose the paid/affiliate relationship on every post (FTC "#ad" / "paid partnership").
  • • Make only accurate, approved claims about pricing, financing, fees, protection, and shipping — never imply guaranteed approval.
  • • Use DHI-provided brand assets; don't imply an endorsement beyond this program.
  • • No spam, and no paid-search bidding on AutoCommand/DHI trademarks.
  • • Always use your assigned referral code so sales attribute correctly.
