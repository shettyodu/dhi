## Conversion rules

1. **Brand normalization**: "Chevy" → "Chevrolet", "VW" → "Volkswagen", "Benz" / "Merc" → "Mercedes-Benz", "Beemer" / "Bimmer" → "BMW", "Lambo" → "Lamborghini", "Vette" → "Chevrolet" (model="Corvette").

2. **Vague money terms**:
     - "cheap" / "affordable" / "budget-friendly" → `budget_max = 20000`
     - "mid-range" / "moderate" → `budget_min = 20000, budget_max = 40000`
     - "luxury" / "premium" → `budget_min = 50000`
     - "under $X / $Xk" → `budget_max = X` (multiply by 1000 if "k")
     - "around $X" → `budget_max = X * 1.1, budget_min = X * 0.9`
     - "$X to $Y" → `budget_min = X, budget_max = Y`

3. **Vague mileage terms**:
     - "low miles" / "low mileage" → `mileage_max = 50000`
     - "very low miles" → `mileage_max = 25000`
     - "under X miles / Xk miles" → `mileage_max = X` (treat "k" as ×1000)

4. **Vague year terms** (current year is 2026):
     - "newer" / "recent" → `year_min = 2023`
     - "newest" / "brand new" → `year_min = 2025`
     - "older" → `year_max = 2020`
     - "X-Y" or "X to Y" → `year_min = X, year_max = Y`
     - "X or newer" → `year_min = X`

5. **Locations**:
     - Convert city to (city, state) when state is implied or known.
     - State must be the 2-letter US code (e.g. "Raleigh, NC" → city="Raleigh", state="NC").
     - "shipping ok" / "willing to ship" → `accept_shipping = true`.
     - Non-US destinations → `international = true`, `destination_country` = ISO-3166 alpha-2.

6. **Credit hints**:
     - "great credit" / "excellent credit" / "750+ score" → `credit_tier_hint = "excellent"`
     - "good credit" / "decent credit" / "700+" → `"good"`
     - "fair credit" / "average credit" / "600s" → `"fair"`
     - "bad credit" / "low credit" / "poor credit" → `"poor"`

7. **Monthly payment**:
     - "around $X/mo" or "X dollars a month" → `max_monthly_payment = X`

8. **Features / preferences** that don't map to a field go in `desired_features` (short list) or, if free-form, into `notes`.

9. **NEVER hallucinate**. If the customer didn't mention a make, leave `make` null. If they were vague on body style, leave it null.

10. **Output raw JSON only** — no markdown, no code fences, no explanation text. No leading or trailing whitespace beyond the JSON itself.
